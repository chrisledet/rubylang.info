    Downloads

  • Using RVM

    Many Rubyists prefer to install Ruby via RVM (Ruby Version Manager). Not only does it make installing Ruby incredibly easy but also allows you to manage other implementations of Ruby such as JRuby, Rubinius, MacRuby, MagLev, & IronRuby.

    RVM is only available for Mac OS X, Linux, or any UNIX-like operating system. Windows users should check out RubyInstaller described in the Windows section below.

  • Mac OS X

    OS X Lion comes with 1.8.7 along with many RubyGems, however, many people tend to use either RVM or Homebrew. To get started with RVM, see the RVM section above. However allows Mac OS X users to easily download and install Unix packages from source. Most installers do this anyway.

  • Windows

    If you’re on Windows, there’s a great project to help you install Ruby called RubyInstaller. It gives you everything you need to set up a full Ruby development environment on Windows.

    To use RubyInstaller, just download it below. Then just use the installer, and you’re done!
